# thsdk 游客模式实测笔记

## 游客模式行为（2026-05-06 实测）

### 能用的功能
| 功能 | 方法 | 状态 | 备注 |
|------|------|------|------|
| 股票搜索 | `search_symbols` | ✓ 正常 | |
| 个股行情 | `market_data_cn` | ✓ 正常，数据完整 | |
| 竞价异动扫描 | `call_auction_anomaly` | ✓ 返回1000条 | 金额字段疑似脱敏（见下方） |
| 实时快讯 | `news` | ✓ 正常 | 默认返回上证指数相关约20条 |
| 问财NLP | `wencai_nlp` | ✓ 可用 | 2026-05-06 确认，查询「今日涨停，非ST」成功返回数据 |
| A股批量行情 | `market_data_cn(列表)` | ✓ 正常 | 同市场可批量 |

### 不稳定/有问题的功能
| 功能 | 方法 | 问题 |
|------|------|------|
| 指数行情 | `market_data_index` | ⚠️ 今日上证指数返回 Empty DataFrame（`[thsdk]QueryData错误:not data`） |
| 集合竞价异动金额 | `call_auction_anomaly` | ⚠️ `总金额` 字段出现 `2147483648`（2^31），疑似脱敏占位符 |

### 游客模式特征
- 每次连接分配随机临时账号（`thsguest_xxxx`）
- 警告：`⚠️ 临时账户可能随时失效，不适合生产环境使用`
- 配置方式：`THS({'username': '...', 'password': '...', 'mac': '...'})`

### 已知限制（游客 vs 正式账号）
| 功能 | 游客 | 正式账号 |
|------|------|---------|
| `wencai_nlp` | ✓ 可用 | ✓ 更稳定 |
| `news()` | ✓ 可用（通用快讯约20条） | ✓ 更完整 |
| 个股精准资讯 | ⚠️ 无，只能靠关键词匹配 | ✓ 支持按股票代码精确拉取 |
| 竞价异动金额 | ⚠️ 部分脱敏 | ✓ 完整 |

## ⚠️ THSCODE 正确值（文档示例有误，2026-05-06 实测修正）

thsdk 文档和代码示例中的 THSCODE 部分错误，必须以实测为准：

### 指数 THSCODE 正确值
| 指数 | 文档示例（❌ 错误） | 实测正确值 | 验证状态 |
|------|-----------------|-----------|---------|
| 上证指数 | `USHI000001` | `USHI1A0001` | ✅ 实测成功 |
| 深证成指 | — | `USZI399001` | ✅ 正常 |
| 创业板指 | — | `USZI399006` | ✅ 正常 |
| 科创50 | `USHI000688` | `USHI1B0688` | ✅ 实测成功 |

验证命令：
```python
import thsdk as _thsdk
with _thsdk.THS(THS_OPS) as ths:
    # 单个调用有效，批量调用返回空
    resp = ths.market_data_index('USHI1A0001')
    print(resp.success, resp.df[['最新价','涨跌幅']].to_string() if not resp.df.empty else 'empty')
```

### market_data_index 批量调用限制
- **批量调用（传列表）**：返回 Empty DataFrame
- **单个调用（逐一）**：正常返回
- **替代方案**：批量时用 `market_data_cn` 对各股票代码逐一查询，或用腾讯行情兜底

## 与同花顺 iFinD API 的关系

- thsdk 是同花顺的 Python SDK，对接 iFinD 数据接口
- iFinD 机构版需付费开通（万元/年起）
- 游客模式是 thsdk 内置的测试模式，不等于免费 iFinD
- 游客模式适合个人临时研究，生产环境建议配置正式账号

## 配置正式账号所需信息

同花顺 iFinD 账号需要：
1. username（手机号/同花顺账号）
2. password
3. mac 地址（绑定的电脑MAC，通常自动获取）

申请渠道：https://www.10jqka.com.cn/ → 产品中心 → iFinD数据接口
